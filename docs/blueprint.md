# **App Name**: Pate Pulse Venture Hub

## Core Features:

- Hero Section with Mission Statement: Display a compelling hero image with Pate Pulse Venture's mission statement, creating an impactful first impression.
- Venture Portfolio Showcase: Showcase portfolio companies/projects, loading data dynamically from Firestore.
- Innovation Lab Highlights: Highlight R&D work and innovative projects.
- Team Bios: Display founder bios with images, creating a personal connection.
- Contact Form with Firestore Integration: A contact form allowing users to send messages, with submissions stored in Firestore.
- Admin Content Tool: Allows authorized personnel to update information about venture portfolios, innovations and team bios using generative AI tool when useful.
- Light/Dark Mode Toggle: Allows users to switch between light and dark color schemes for optimal viewing experience.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to convey innovation and sophistication.
- Background color: Light gray (#F0F0F0), nearly white, for a clean, modern backdrop.
- Accent color: Sky blue (#87CEEB), which complements the indigo and adds a touch of vibrancy. #4B0082 converted to hex from HSL 275,100,25; #F0F0F0 converted from HSL 0,0,94; #87CEEB converted from HSL 203,71,75.
- Font pairing: 'Inter' (sans-serif) for headlines and body text. Note: currently only Google Fonts are supported.
- Grid-based layout with balanced whitespace, ensuring clean alignment and readability.
- Subtle motion effects on hover and scroll, inspired by Visvik.n.
- Minimalist icons that represent venture categories and team member roles.
